Calculates the filtered version of the raw input data AND creates filter

Parameters
[in]matDataInThe data which is to be filtered
[in]typeof the filter: LPF, HPF, BPF, NOTCH (from enum FilterType)
[in]iOrder
[in]dCenterfreqdetermines the center of the frequency
[in]dBandwidthignored if FilterType is set to LPF,HPF. if NOTCH/BPF: bandwidth of stop-/passband
[in]dTransitiondetermines the width of the filter slopes (steepness)
[in]dSFreqsampling frequency
[in]vecPicks- used channel as index in QVector
[in]iOrderrepresents the order of the filter, the higher the higher is the stopband attenuation
[in]iFftLengthlength of the fft (multiple integer of 2^x) - Default = 4096
[in]designMethodspecifies the design method to use. Choose between Cosind and Tschebyscheff; Defaul = Cosine
Returns
The filtered data in form of a matrix.
