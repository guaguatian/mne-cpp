class  MNEBROWSE::RawDelegate
 

Detailed Description

This class represents the delegate of the model/view framework of mne_browse application. Since it is derived from the QAbstractItemDelegate class [1], the virtual functions paint() und sizeHint() need to be reimplemented. The paint() function is invoked from the connected QTableView for each table cell for any individual item with a certain QModelIndex. The task of this function is to paint this cell with the data that is requested by index.model()->data(index,Qt::DisplayRole); with the corresponding QModelIndex.
